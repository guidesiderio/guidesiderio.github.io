Sobre mim

Graduando em Ciência da Computação pela UFPI, com formação prevista para 2028.1. Atuo em dois estágios simultâneos: no Tribunal de Justiça do Piauí (TJPI), trabalho com BI e Governança de Dados — extraio dados via SQL no contexto judiciário, desenvolvo dashboards institucionais no Qlik Sense, monitoro métricas e crio indicadores para tomada de decisão, participando do pipeline de dados de ponta a ponta. Na Pró-Reitoria de Administração da UFPI (PRAD), criei o primeiro dashboard da unidade e tenho liberdade para atuar também em desenvolvimento web, como a criação de landing pages. Em paralelo, mantenho uma capacitação contínua em Ciência de Dados, com aprofundamento em Cloud (AWS) voltado à análise de dados e preparação para certificação, além de estudos em IA Generativa, LLMs, Code Agents e Business Intelligence.

TJPI · 2026

Dashboard das Salas de Audiência — CEJUSC I

Painel desenvolvido no Qlik Sense para monitoramento da ocupação das salas presenciais e virtuais do CEJUSC I. Utiliza set analysis sobre o banco de dados do PJe 1º Grau (PostgreSQL) para construir indicadores de taxa de ocupação, acordos por audiência e desempenho de mediadores. Demanda originada pela coordenação do CEJUSC I para otimizar o uso das salas físicas.

SQL PostgreSQL Qlik Sense Set Analysis
Institucional

TJPI · 2026

Dashboard de Partes sem CPF/CNPJ

Painel desenvolvido no Qlik Sense a partir de solicitação formal da Corregedoria-Geral de Justiça. Identifica feitos de execução fiscal sem indicação de CPF ou CNPJ dos executados e, nessas hipóteses, sugere minuta de extinção do processo. Dados extraídos via SQL do banco de dados do PJe (PostgreSQL).

SQL PostgreSQL Qlik Sense

Dashboard de Complementos Inválidos

Painel para monitoramento da qualidade dos dados enviados ao DataJud, com foco no Indicador A3 — Percentual de Complementos Válidos, do Painel de Saneamento do Conselho Nacional de Justiça (CNJ). Auxilia na identificação de inconsistências e na tomada de decisão para correção de processos com complementos de movimentos inválidos.
